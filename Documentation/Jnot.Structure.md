# Jnot.Structure.md

## John's Nifty Office Tools - File Structure

- [ExcelAddIn](ExcelAddIn.csproj.md) → Project that interacts with Excel
- [ExcelAddIn.Interop](ExcelAddin.Interop.csproj.md) → Project that interacts with Excel files
- [FileRenamer](FileRenamer.csproj.md) → Provides code to take Excel pivot table files and rename them according to defineable patterns
- [FileRenamer.Tests](FileRenamer.Tests.csproj.md) → Tests for FileRenamer
- [Shared.Config](Shared.Config.csproj.md) → Code to read/write configuration files for Jnot.*
- [Shared.Config.Tests](Shared.Config.Tests.csproj.md) → Tests for Shared.Config
- [Shared.Info](Shared.Info.csproj.md) → Code to provide metainformation about files for Jnot.*
- [Shared.UI.Controls](Shared.UI.Controls.csproj.md) → Shared NET 10 WinForms Controls
- [Shared.UI](Shared.UI.csproj.md) → Shared NET 10 Winforms (other than controls)
- [ThrowAway.Demo](ThrowAway.Demo.csproj.md) → Demo of Shared.UI + other Jnot.* features
